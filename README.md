## 포트폴리오

### 사유

포트폴리오를 요구하는 회사가 많아졌고, 사용하고 싶었던 기술이나 라이브러리를 개인의 프로젝트에서 사용해보고 싶었습니다.

따라서 가장 자신있는 웹으로 페이지를 꾸려 취업준비와 공부를 동시에 진행했습니다.

### 출처

디자인을 고민하는데 많은 시간을 쏟고 싶지 않았고 옛날부터 많이 참고했던 사이트를 참고해서 만들었습니다. 혹시나 문제가 된다면 그 즉시 내리도록 하겠습니다.
